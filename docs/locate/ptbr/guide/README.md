## Aviso
1. As versões mobile à seguir são hybrid-app e a versão native, essas duas versões precisam de nodejs junto com as demais dependênciais para funcionar.
2. O processo de build de cada uma dessas versões funciona de modo diferente.
3. A versão  hybrid-app serve para hospedagem enquanto a versão native apenas funciona após ser gerado um apk para android ou apk em ios.
4. Você pode hospedar a pasta mobile, web, docs em qualquer hospedagem, nuvem ou site. O mais indicado são plataformas como Vercel, Netlify ou GitHub Pages por serem gratuitas e com ótimos recursos na conta gratuita caso queira testar essa ideia como mvp(minimal value product).
